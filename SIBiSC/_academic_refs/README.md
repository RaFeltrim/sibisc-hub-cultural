# Academic References

Esta pasta reúne materiais acadêmicos e instruções externas usados pelo time durante a disciplina.

## Conteúdo

- slides da professora
- PDFs de instrução recebidos ao longo do projeto

## Como Usar

- consultar este material para alinhar o projeto com a disciplina
- registrar interpretações e decisões finais em `SIBiSC/docs`
- evitar usar PDFs externos como substitutos da documentação oficial

## Regra

Esses materiais são referências de apoio. As interpretações, decisões e adaptações válidas do projeto devem ser registradas em `SIBiSC/docs`.
