# Document PDFs

Esta pasta é reservada para os **espelhos em PDF** da documentação oficial mantida em Markdown.

## Objetivo

Facilitar leitura, compartilhamento e consulta rápida da documentação do projeto sem substituir a fonte oficial.

## Regra

- a fonte oficial continua sendo `SIBiSC/docs/*.md`
- PDFs aqui devem espelhar documentos oficiais do projeto
- materiais acadêmicos ou PDFs externos ficam em `SIBiSC/_academic_refs`
- quando um `.md` relevante mudar, o PDF correspondente deve ser regenerado
