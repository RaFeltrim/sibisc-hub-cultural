# Como Contribuir

## Fluxo Esperado

1. alinhar a mudança com backlog ou decisão registrada
2. atualizar documentação relevante
3. implementar a alteração
4. validar impacto com TL e QA quando necessário
5. abrir PR com contexto e links

## O Que Toda Contribuição Deve Informar

- problema que está resolvendo
- arquivo ou área impactada
- documento atualizado
- risco ou dependência remanescente

## Regra

Contribuição técnica sem documentação correspondente não é considerada completa.
