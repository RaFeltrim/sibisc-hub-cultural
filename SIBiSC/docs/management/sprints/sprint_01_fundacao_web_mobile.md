# Sprint 01 - Fundacao Web e Mobile First

## Objetivo

Construir a base de interface e navegacao para que o time veja o produto nascer cedo, com foco em responsividade e estrutura reutilizavel.

## Entregas da Sprint

- layout base
- navegacao principal
- paginas iniciais dos modulos
- estados de loading, erro e vazio reutilizaveis

## Cards Recomendados

- `T-BASE-003`
- preparacao visual para `T-NOT-001`
- preparacao visual para `T-EVT-001`
- preparacao visual para `T-ACV-001`

## O Que os Alunos Aprendem

- componentes de interface
- navegacao entre telas
- conceito de mobile first
- diferenca entre pagina, componente e estado de tela

## Criterio de Sucesso

- navegacao funcionando entre os modulos
- leitura confortavel em desktop e mobile
- base pronta para plugar dados reais sem refazer a estrutura
