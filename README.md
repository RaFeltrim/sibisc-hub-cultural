# Web e Mobile

Repositorio principal do projeto **SIBiSC**, organizado para concentrar o produto, a documentacao oficial e os artefatos academicos de apoio de forma clara e escalavel.

## Estrutura

- `SIBiSC/`: produto principal, documentacao oficial, codigo e integracoes futuras.

- `SIBiSC/_academic_refs/`: slides e referencias academicas usadas pelo time.
