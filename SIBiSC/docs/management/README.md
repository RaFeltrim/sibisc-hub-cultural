# Gestao do Projeto SIBiSC

## Onde Esta a Pasta de Gestao

A pasta de gestao do projeto esta em:

- `SIBiSC/docs/management/`

Ela concentra os documentos de planejamento, organizacao de sprint, riscos, atas e estrutura recomendada para espelhar o trabalho no Jira.

## Como Navegar

Se voce quer entender o projeto rapido, siga esta ordem:

1. [`roadmap.md`](./roadmap.md)
2. [`plano_mvp_jira.md`](./plano_mvp_jira.md)
3. [`sprints/README.md`](./sprints/README.md)
4. documentos tecnicos e de QA referenciados em `SIBiSC/docs/INDEX.md`

## Leitura para Alunos

Para os alunos que ainda estao aprendendo, o mais importante e entender que o projeto nao deve ser atacado inteiro de uma vez. O trabalho foi quebrado em partes pequenas para permitir:

- entendimento rapido do objetivo
- entregas menores e mais seguras
- aprendizado progressivo em Web e Mobile
- menos risco de incompatibilidade entre pecas

## Regra de Ouro

Antes de criar um card no Jira ou iniciar uma implementacao, confirmar:

- em qual sprint esse item entra
- a qual US ele pertence
- qual e a task principal
- quais subtasks menores tornam a entrega executavel
- qual documentacao precisara ser atualizada

## Resultado Esperado

Esta pasta deve permitir que uma pessoa nova no projeto consiga responder rapidamente:

- o que vamos construir ate o MVP
- em que ordem vamos construir
- como quebrar isso em cards no Jira
- como saber se a entrega esta pronta para integrar
