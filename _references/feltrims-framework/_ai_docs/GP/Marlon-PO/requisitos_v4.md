# 📘 V4.0: Gems e Automação (A Chegada da IA)

**Data Média:** Fim de 2025
**Autor:** PM & PO
**Epics Focadas:** Integração de Inteligência Artificial nativa.

## O Cenário
Passamos a utilizar a API da Qwen Max. Com isso, títulos ruins deixaram de existir. As descrições começaram a abraçar SEO e Gatilhos Mentais.
- A dor principal tornou-se "A Qualidade dos Dados Entrantes".
- Para isso criamos O *Gatekeeper*.

## Regras Implementadas na V4
1. Proibição de Unicode especial nos títulos.
2. Sanitização estrita do formato numérico EAN-13 (GS1 Base).
3. Remoção de blocos "Aviso de Baterias" da Amazon quando enviados para o Mercado Livre.
