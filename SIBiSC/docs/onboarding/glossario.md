# Glossário

- `SIBiSC`: Sistema Integrado de Bibliotecas de São Carlos
- `MVP`: versão mínima viável do produto
- `RLS`: Row Level Security
- `PWA`: Progressive Web App
- `TL`: Tech Lead
- `CIAO`: Chief Intelligence and AI Officer
- `Ingestão`: processo de coletar e consolidar dados de fontes externas
- `Curadoria`: revisão humana de conteúdo ou dado antes da publicação
- `Fallback`: comportamento alternativo quando um recurso principal falha ou não está disponível
- `Enabler`: item técnico necessário para permitir funcionalidades de negócio
