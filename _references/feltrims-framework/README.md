# Feltrims Framework Reference

Esta pasta preserva o material do **Feltrims Framework** como referência versionada para o projeto.

## Como Usar

- consultar padrões, papéis e rituais
- adaptar apenas o que fizer sentido para o contexto do SIBiSC
- registrar a adaptação final em `SIBiSC/FRAMEWORK.md` e `SIBiSC/docs`

## Importante

O framework não é o produto. O produto principal deste repositório é o **SIBiSC**.
